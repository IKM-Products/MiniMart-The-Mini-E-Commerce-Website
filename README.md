**MiniMart** is a simple and responsive mini e-commerce website designed to provide users with an easy online shopping experience. The platform allows users to browse products, view details, add items to the cart, and simulate an online purchasing system through a clean and user-friendly interface.

## 🚀 Features

* 🔎 **Product Search & Categories** - Easily browse and search products by category.

* 🛒 **Shopping Cart System** - Add and remove products from the cart dynamically.

* 📱 **Responsive Design** - Optimized for desktop and mobile devices.

* ⚡ **Simple & Fast User Interface** - Lightweight design for smooth performance and usability.

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Code Editor:** Visual Studio Code
* **Images:** Unsplash API

## 📱 How It Works

1. User visits the website.
2. Products are displayed on the homepage.
3. User can:
   * Search or browse products
   * Add items to cart
4. Cart updates dynamically based on user actions.
5. User proceeds to checkout simulation.

## 🎯 Objective

The main objective of **MiniMart** is to demonstrate the core functionalities of an e-commerce platform while providing a simple and interactive shopping experience.
